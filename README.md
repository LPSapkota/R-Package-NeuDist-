# NeuDist: Univariate Continuous Distributions with Model Diagnostics

NeuDist is an R package that implements univariate continuous probability distributions and provides model diagnostics.  
It supports distributions based on the Lindley, Logistic, Half-Cauchy, Half-Logistic, and Poisson families, along with tools for model evaluation.

## 📖 Overview
- Functions for probability density, cumulative distribution, quantile, and hazard evaluation.
- Random variate generation for supported distributions.
- Diagnostic procedures including:
  - Q-Q and P-P plots
  - Goodness-of-fit tests
  - Model selection criteria

## 📂 Repository Contents
- Core R functions for distribution evaluation.
- Scripts for diagnostic plots and model comparison.
- Documentation and metadata for package installation.

## ⚙️ Requirements
- R (≥ 3.5)
- Imports: `stats`, `graphics`, `gofTest`

Install dependencies in R:

install.packages("gofTest")

install.packages("NeuDist")
